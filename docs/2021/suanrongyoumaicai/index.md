# 蒜蓉油麦菜

## 1、食材准备

主料：油麦菜、小米椒

辅料：油、蒜末

调料：生抽、白糖、盐、鸡精

## 2、食材处理

油麦菜：切两段，不要太短；洗干净的菜，一定要水沥干，否则加盐会出比较多的水；

小米椒：切小段，0.5 cm 左右；

## 3、烹制步骤、注意事项

爆香蒜蓉：油烧热，下蒜蓉小米椒爆香，不要炒太久，10s 即可；

炒油麦菜：下油麦菜，慢慢将油麦菜炒稍微开始变瘪，开始下料；

下料调味：加入生抽、糖少量、盐、鸡精，翻炒均匀即可出锅，不要炒过；
